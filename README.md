# retail-schedule-manager
A personal project for a retail based schedule manager. This application allows managers to create templates for employees' schedules, save those templates, and pull them back up once needed. Also, allows users to input employee specific data, ie. Name, booked days off, position ect.


**PLAN OF ACTION**

*SCHEDULE MANAGER PROJECT*
|
|_\ APPS
    |_> Employees App -> Used to EDIT, ADD, REMOVE and view employees. Also to add booked days off
    |
    |_\ Schedule App -> Used to build schedule on weekly basis, edit schedules, and save schedules for later use
    |
    |_\Saved Schedule App -> Used to view and pull to edit saved schedules
